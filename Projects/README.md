# Multi Sensor Reading and Processing 

## Sensirion CO2 Sensor
Power Supply  
3,3 ... 5,0 V -> Vdd-Pin

I2C Addresses
- 0x62

## CCS811 Sensor Module
Power Supply  
1,8 ... 3,6 V -> Vcc-Pin

I2C Addresses
- 0x5A -> ADD-Pin Low
- 0x5B -> ADD-Pin High

## ENS160+AHT21 CO2 Room Temperature Module
Power Supply  
5 V -> VIN-Pin
3,3 V -> 3V3-Pin

I2C Addresses  
- ENS160
  - 0x52 -> ADD Pin LOW
  - 0x53 -> ADD Pin HIGH 
- AHT21
  - 0x38
